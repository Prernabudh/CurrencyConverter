# Steps to run the project

## Step 1: Install node modules

```bash
# using npm
npm install

# OR using Yarn
yarn
```

## Step 2: Install pod

```bash
cd ios
pod install
cd ..
```

## Step 3: Start your Application

### For Android

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

### For iOS

```bash
# using npm
npm run ios

# OR using Yarn
yarn ios
```
